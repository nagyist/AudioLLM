## 🤝 Contributing

Add an `items/<Abbreviation>.json` (template in [`schema.json`](schema.json)), run `python3 format_input.py` to regenerate the README and site data, and open a PR. CI validates JSON, checks README sync, and the site rebuilds automatically on merge. Full guide: **[CONTRIBUTING.md](CONTRIBUTING.md)**.

Don't want to write a PR yourself? **[Suggest a paper](https://github.com/BinWang28/audio-ai-hub/issues/new?template=add-paper.yml)** via the issue form and a maintainer will add it.
